# Networking Foundation Summary

Finished the networking foundation phase.

Topics covered:

* IP Addresses
* Public vs Private IPs
* DNS
* Ports
* TCP & UDP
* NAT
* ARP
* Routing
* DHCP
* HTTP & HTTPS
* VPN Basics
* OSI Model
* Switching

Things I found interesting:

* DNS is basically the Internet's phonebook.
* TCP and UDP solve different problems.
* ARP exists because devices need MAC addresses to communicate on a local network.
* Routers don't know the whole Internet. They just know the next router to send traffic to.
* HTTPS adds encryption to web traffic.

The most interesting thing I learned was what actually happens behind the scenes when visiting a website.
